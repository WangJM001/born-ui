## API

| 属性         | 描述                         | 类型                  | 默认值 |
| ------------ | ---------------------------- | --------------------- | ------ |
| colorful     | 根据 `flag` 显示不一样的颜色 | `boolean`             | true   |
| flag         | 类型                         | `up \| down`          | true   |
| reverseColor | `up` `down` 颜色反转         | `boolean`             | false  |
| className    | className                    | `string`              | -      |
| style        | 样式                         | `React.CSSProperties` | -      |
