## API

> 这里只列出与 antd card 不同的 api

| 属性        | 描述                                                   | 类型      | 默认值 |
| ----------- | ------------------------------------------------------ | --------- | ------ |
| headerSplit | 是否展示 header 分割线                                 | `boolean` | false  |
| ghost       | 幽灵模式，即是否取消卡片 header 的 padding 和 背景颜色 | `boolean` | false  |
| bordered    | 是否展示边框                                           | `boolean` | false  |
